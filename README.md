# FCLM Tampermonkey Scripts

Floating rate tracker overlays for [fclm-portal.amazon.com](https://fclm-portal.amazon.com).

---

## 📊 RPND Rate Tracker

Tracks **Decant, Decant Non-TI, Prep, and Each Receive** UPH in real time.

**[⬇ Install RPND Rate Tracker](https://raw.githubusercontent.com/tytyh/tampermonkey-scripts/main/fclm-rate-tracker.user.js)**

---

## 📦 UIS / ManSort Tracker

Tracks **UIS 5LB, UIS 20LB, and ManSort** UPH in real time.

**[⬇ Install UIS / ManSort Tracker](https://raw.githubusercontent.com/tytyh/tampermonkey-scripts/main/fclm-uis-mansort-tracker.user.js)**

---

## Setup

1. Install [Tampermonkey](https://www.tampermonkey.net/) in Chrome
2. Click an install link above — Tampermonkey will prompt you to install
3. Navigate to `fclm-portal.amazon.com` — the overlays will appear automatically
4. Use the ⚙️ gear icon on each panel to set your shift window and targets

## Notes

- Both panels are draggable and collapsible
- Auto-refreshes every 60 seconds
- Configured for **RFD2**
